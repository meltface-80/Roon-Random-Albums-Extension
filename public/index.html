      How to install librespot

      librespot is the open-source Spotify Connect receiver. Install it on the same machine as this extension, then restart the extension.

      Option 1 — Debian / Ubuntu / DietPi / Raspberry Pi OS (easiest)

      apt-get install librespot

      If the package is not found or your Raspberry Pi needs a newer build, install via raspotify instead:

      curl -sSL https://dtcooper.github.io/raspotify/install.sh | sh
# Then stop the raspotify service — we only need the binary:
systemctl stop raspotify
systemctl disable raspotify

      The librespot binary will be at /usr/bin/librespot.

      Option 2 — macOS (Homebrew)

      brew install librespot

      Option 3 — Build from source (any platform)

      First install or update Rust via rustup.rs — the system Rust on Debian/DietPi is too old:

      curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh -s -- -y
source ~/.cargo/env
cargo install librespot

      ⚠️ Compiling takes 20–40 minutes on Raspberry Pi hardware.

      Custom path

      If librespot is installed somewhere non-standard, set the environment variable before starting the extension:

      LIBRESPOT_PATH=/path/to/librespot node index.js
